## 一、方案标题
涂鸦宠物喂食器
## 二、方案应用场景
地点：用户家中  
功能：可以自动添加食物，也可以远程手动添加食物；可以检测宠物食物余量，返回到用户手机；还能远程打开小夜灯，故障报警等。
